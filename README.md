Plataforma para gestionar series de forma simple y moderna
NextFLIX nace como proyecto práctico para aplicar arquitectura con Next.js, consumo de APIs reales y validación de formularios.

## Stack
Construido con Next.js, componentes reutilizables con Radix UI y validaciones robustas con Valibot.


consume la API https://nestflix.onrender.com/series


## Correr el proyecto

Primero renombra el archivo .env.example a .env

Luego ejecuta en la terminal los siguientes comnandos:
```bash
npm install
npm run dev
```
## Visualizar el proyecto
Abre [http://localhost:3000](http://localhost:3000) en tu navegador

## Deploy en Vercel

El proyecto ha sido subido a Vercel

Ingresa a esta direccion: [https://series-nextjs.vercel.app/](https://series-nextjs.vercel.app/)
